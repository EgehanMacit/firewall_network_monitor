# NetGuard - Firewall & Monitor 🛡️

is a professional Windows utility that provides real‑time visibility into firewall status and network activity. Built with Kivy and psutil, it offers a clean, responsive interface for monitoring firewall profiles, tracking live bandwidth, and auditing active connections and processes. Designed for IT professionals and power users, NetGuard combines continuous monitoring, detailed logging, and process-level context to help secure and troubleshoot Windows systems.
Key Features
- Firewall Profile Monitoring: Continuously reports the status of Domain, Private, and Public firewall profiles.
- Real‑Time Bandwidth Tracking: Displays live upload and download rates in KB/s with smooth, low‑latency updates.
- Active Connection Overview: Lists current network connections with remote endpoints, local ports, and connection states.
- Process Correlation: Maps connections to PIDs and process names for rapid attribution and investigation.
- Connection History & Logging: Maintains a searchable log of past connections, timestamps, and associated processes for auditing and forensics.
- Lightweight and Cross‑Platform UI: Modern Kivy interface optimized for responsiveness and minimal resource overhead.
NetGuard is ideal for administrators who need an unobtrusive, accurate tool to monitor firewall posture and network activity on Windows endpoints.
